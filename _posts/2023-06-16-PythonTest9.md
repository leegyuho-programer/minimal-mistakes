1번 지표는 튜브형(T)보다 라이언형(R)의 점수가 더 높습니다. 따라서 첫 번째 지표의 성격 유형은 R입니다.
하지만, 2, 3, 4번 지표는 모두 0점으로 동일한 점수입니다. 따라서 2, 3, 4번 지표의 성격 유형은 사전순으로 빠른 C, J, A입니다.

문제 풀이
def solution(surveys, choices):
    result = {i:0 for i in "RTFCMJAN"}
    for survey, choice in zip(surveys, choices):
        #survey front ++
        if choice < 4:
            result[survey[0]] += 4 - choice
        #survey back ++
        elif choice > 4:
            result[survey[1]] += choice - 4
    answer = ""
    for category in ["RT", "CF", "JM", "AN"]:
        if result[category[0]] < result[category[1]]:
            answer += category[1]
        else:
            answer += category[0]
    return answer

문제 풀이
def solution(today, terms, privacies):
    ty, tm, td = map(int, today.split("."))
    today = ty * (12 * 28) + tm * 28 + td
    t2m = {term.split()[0]:int(term.split()[1]) for term in terms}
    answer = []
    for idx, privacy in enumerate(privacies):
        sday, term = privacy.split()
        sy, sm, sd = map(int, sday.split("."))
        sday = sy * (12 * 28) + sm * 28 + sd
        termday = t2m[term] * 28
        if not today < sday + termday:
            answer.append(idx+1)
    return answer
